ProfStats
========
[![GitHub](https://img.shields.io/github/downloads/NoaZetterman/prof-stats/total?logo=github)](https://github.com/NoaZetterman/prof-stats/releases)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](https://github.com/NoaZetterman/prof-stats/blob/master/LICENSE)

ProfStats is a Wynncraft mod that tracks profession data locally. It tracks both gathering data and crafting data for all gathers and crafts. The data can be found in the `data/prof-stats/` folder in the minecraft instance folder that the mod is active in.

ProfStats supports Minecraft 1.21.4, and uses **Fabric**. 

Download a release
========
You can download the latest build from the [releases page](https://github.com/NoaZetterman/prof-stats/releases).
